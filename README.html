# Zomercompetitie Open-source FastAPI webapp voor het beheren van een dartscompetitie: - aanwezigheden per speelavond - poules met anti-herhaling op basis van historische ontmoetingen - knock-out vanaf halve finales of kwartfinales (afhankelijk van opkomst) - bulk invoer van uitslagen + highlights (180, 100+, ≤15) - seizoensoverzicht en archivering - mobielvriendelijke UI + PWA ondersteuning ## Tech stack - Python 3.10+ - FastAPI + Jinja2 - SQLAlchemy - SQLite - Nginx + systemd (via install script) ## Snelle lokale start ```bash python3 -m venv .venv source .venv/bin/activate pip install -e . uvicorn main:app --reload ``` Open daarna: `http://127.0.0.1:8000` --- ## Installatie op Ubuntu server Gebruik het install script vanuit de repo root: ```bash bash ./scripts/install_ubuntu.sh ``` ### Belangrijkste omgevingsvariabelen - `DOMAIN` – domeinnaam voor Nginx/TLS (bijv. `mijn-darts-app.nl`) - `TLS_EMAIL` – e-mailadres voor Let's Encrypt - `ENABLE_TLS` – `1` om TLS in te schakelen, anders `0` - `TLS_MODE` – `http` of `dns` - `DNS_PROVIDER` – bij DNS mode: `cloudflare`, `digitalocean`, `route53` - `DNS_CREDENTIALS_FILE` – pad naar credentials bestand (indien nodig) - `ENABLE_ONTWIKKELTOOLS` – `true` of `false` (admin testtools tonen/verbergen) - `ENABLE_UPDATE_CHECK` – `true` of `false` (GitHub updatecontrole) - `GITHUB_REPOSITORY` – repo slug voor update checker, bijv. `owner/repo` ### Voorbeeld: productie-installatie met HTTPS ```bash DOMAIN=mijn-darts-app.nl \ TLS_EMAIL=admin@mijn-darts-app.nl \ ENABLE_TLS=1 \ TLS_MODE=http \ ENABLE_ONTWIKKELTOOLS=false \ ENABLE_UPDATE_CHECK=true \ GITHUB_REPOSITORY=owner/zomercompetitie \ bash ./scripts/install_ubuntu.sh ``` ### Voorbeeld: DNS validatie (Cloudflare) ```bash sudo mkdir -p /root/.secrets/certbot sudo tee /root/.secrets/certbot/cloudflare.ini >/dev/null <<'CF' dns_cloudflare_api_token = CF sudo chmod 600 /root/.secrets/certbot/cloudflare.ini DOMAIN=mijn-darts-app.nl \ TLS_EMAIL=admin@mijn-darts-app.nl \ ENABLE_TLS=1 \ TLS_MODE=dns \ DNS_PROVIDER=cloudflare \ DNS_CREDENTIALS_FILE=/root/.secrets/certbot/cloudflare.ini \ bash ./scripts/install_ubuntu.sh ``` --- ## Updaten ```bash git pull bash ./scripts/install_ubuntu.sh sudo systemctl restart zomercompetitie sudo systemctl reload nginx ``` ## Data locatie Standaard databasepad in productie: `/var/lib/zomercompetitie/zomercompetitie.db` ## Licentie Kies zelf je gewenste licentie voor publicatie (bijv.
